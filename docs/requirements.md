# Forge AI – Requirements

## Required (runtime)

- C++17 compatible compiler (g++ ≥ 11, clang ≥ 14)
- CMake ≥ 3.16
- make
- Unix domain socket support

## Optional (development & testing)

- socat (for socket testing)
- git

## Libraries

- nlohmann/json (header-only)
- llama.cpp (vendored later)

## Supported OS

- Linux (Ubuntu 22.04+)
- macOS (Apple Silicon & Intel)
